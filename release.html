Release Notes

What's new in Comma.

v0.0.1 Internal Beta March 17, 2026

The first internal beta of Comma — a macOS-native voice input tool. Press one key, start speaking, and your words appear as text in any app.

✨ Core Features

  • 🎤 Voice Dictation — Press fn to start, press fn again to stop. Text appears instantly in any app.
  • 🌐 Real-time Translation — fn + Shift: speak Chinese, get English text. Or vice versa.
  • ✨ AI Polish — Auto-corrects punctuation, removes filler words, fixes formatting.
  • 📖 Custom Dictionary — Add names, brands & technical terms so ASR never mishears them.
  • 📊 Usage Dashboard — Track words dictated, time saved, and accuracy trends.
  • 🔒 Secure & Private — Voice data is processed securely via cloud ASR. We never store your audio.

🏗 Under the Hood

  • ◆ macOS-native — Built entirely in Swift, designed for macOS 12.0+.
  • ◆ ASRProvider protocol — Swap between on-device (Apple SFSpeechRecognizer) and cloud ASR.
  • ◆ Text Injection — Via Accessibility API, with clipboard fallback for Electron apps.
  • ◆ Toggle + Hold — 400ms threshold auto-detects your intent.
  • ◆ Privacy by design — All data stored locally via SwiftData.
  • ◆ 174 tests, 25 suites — TDD-driven development from day one.

📦 Install

  • 1. Download the .dmg from the internal distribution channel.
  • 2. Drag Comma.app into your Applications folder.
  • 3. Launch Comma — the onboarding guide will walk you through permissions and setup.

⚠️ Known Limitations (v0.0.1)

This is an internal-only beta. Some features may be incomplete or rough around the edges. Please report any issues to the team. Your feedback helps shape what Comma becomes.
